# 🏠 House Price Dashboard

## 📌 Overview

This project is an interactive Power BI dashboard for analyzing house prices using real-world data.
It helps understand how different property features and amenities affect pricing.

---

## ⚙️ Tools Used

* Power BI
* Dataset: Housing.csv

---

## 📊 Features

* Total Houses and Average Price (₹ Lakhs)
* Percentage of houses with AC
* Percentage of houses on main road
* Filter by furnishing status
* Price comparison across different property features
* Furnishing distribution using donut chart

## 💡 Insights

* Prices vary based on property features
* Furnished houses are generally more expensive
* Houses on main roads tend to have higher prices
* Amenities like AC increase property value
